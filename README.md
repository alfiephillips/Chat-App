# Chat-App

Chat-App is a simple messaging app using SocketIO and Flask.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install the dependencies needed in requirements.txt

```bash
pip install -r requirements.txt
...
...
python main.py
```
